# ✈️ Aeroplane Management System

This is a simple **Aeroplane Management System** built with Python.  
It allows users to:
- ➕ Add new aeroplanes  
- 🧑‍🤝‍🧑 Add passengers to an existing aeroplane  
- 👀 View the passenger list for an existing aeroplane
